# Self-Debate Experiment v4

Tests whether a structured self-debate protocol — where an adversarial Critic and Defender each independently assess an ML evaluation scenario before an Orchestrator adjudicates — produces higher-quality verdicts than a single-pass baseline on a 53-case benchmark of common ML evaluation failures. Five conditions are compared: `isolated_debate` (role-separated parallel assessment), `multiround` (Defender sees Critique; up to 4 rounds), `forced_multiround` (minimum 2 rounds; hard cases only), `ensemble` (3 independent assessors + synthesizer), and `baseline` (single-pass). The primary metric is fair-comparison lift on IDR, IDP, DRQ, and FVC — dimensions where baseline has equal structural agency.

## Scope Exclusions

- Cases not generated by this experiment. All benchmark cases were generated by a non-Anthropic LLM and validated by CASE_VERIFIER before any experiment agent saw them.
- Cross-vendor scoring requires an external model API key (Phase 9 only).
- No API key required for Phases 0–8, 10–11. Runs inside an authenticated Claude Code session.
- Mechanistic attribution for multiround vs. isolated_debate: the two conditions are not cleanly isolated (see EXECUTION_PLAN.md). Results are reported descriptively.
